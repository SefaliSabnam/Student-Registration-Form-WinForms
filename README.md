# Student Registration Form (Windows Forms)

A simple Student Registration Form developed using **C# Windows Forms (.NET Framework)**.

## Features

- Upload profile photo using PictureBox
- Browse image using OpenFileDialog
- Enter student name
- Select Date of Birth
- Select Gender
- Select Marital Status
- Select Qualification
- Choose Experience from ComboBox
- Write About Project using RichTextBox
- Display all entered details using MessageBox

## Technologies Used

- C#
- Windows Forms (.NET Framework)
- Visual Studio 2022

## Controls Used

- Label
- TextBox
- PictureBox
- Button
- DateTimePicker
- RadioButton
- CheckBox
- ComboBox
- RichTextBox
